# SQL

A comprehensive collection of SQL scripts covering everything from basic concepts to advanced database techniques. Perfect for beginners and intermediate developers looking to master SQL.

## 📋 Overview

This repository contains **100+ well-organized SQL scripts** that demonstrate practical SQL usage across various topics:

- DDL (Data Definition Language)
- DML (Data Manipulation Language)
- Joins and Subqueries
- Window Functions
- CTEs (Common Table Expressions)
- Transactions
- Indexing & Optimization
- Real-world schema examples and queries

## 📁 File Structure

Files are numbered and organized by topic:

### **Core SQL Concepts**
- **001-003**: Table Creation, ALTER, DROP & TRUNCATE
- **004-008**: Indexes, Views, Stored Procedures, Functions, Triggers
- **009-012**: INSERT, UPDATE, DELETE, MERGE/UPSERT

### **Querying Data**
- **013-016**: SELECT, WHERE, ORDER BY, GROUP BY, HAVING
- **017-023**: All types of JOINs (INNER, LEFT, RIGHT, FULL, CROSS, SELF)
- **024-028**: Subqueries and EXISTS/NOT EXISTS
- **029-033**: Aggregation, UNION, INTERSECT, EXCEPT

### **Advanced Features**
- **034-040**: Window Functions (ROW_NUMBER, RANK, LEAD/LAG, etc.)
- **039-040**: Recursive & Non-recursive CTEs
- **041-050**: String, Date, and Conditional functions
- **051-055**: Constraints (PK, FK, UNIQUE, CHECK, etc.)
- **056-058**: Transactions & Savepoints

### **Performance & Optimization**
- **059-061**: EXPLAIN, Indexing Strategies, Query Optimization

### **Real-World Examples**
- **062-063**: E-commerce Schema + Queries
- **064-065**: Library Management System
- **066-067**: School Database
- **068-069**: Banking System
- **070-071**: Employee Management
- **072-073**: Hospital Management

### **Advanced Topics**
- Running Totals, Moving Averages, Year-over-Year Analysis
- Pivot & Unpivot
- Dynamic SQL
- Temporal Tables
- Data Quality & Deduplication
- Report Generation

## 🚀 How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/AsifOntuHasan/SQL.git
   ```

2. Run scripts in order (recommended for beginners) or jump to specific topics.

3. Use with any SQL database:
   - **PostgreSQL** (most scripts)
   - **SQL Server** / **MySQL** / **SQLite** (some syntax variations may apply)
